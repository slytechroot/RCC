The sed command is also able to insert text after a pattern. For this type of expression, use an insert change with the a\ expression.

/PATTERN/a\TEXT\

To append the string End of report in the /etc/hosts file following the line containing allrouters, execute the following command:

sed '/allrouters/a\End of report' /etc/hosts
