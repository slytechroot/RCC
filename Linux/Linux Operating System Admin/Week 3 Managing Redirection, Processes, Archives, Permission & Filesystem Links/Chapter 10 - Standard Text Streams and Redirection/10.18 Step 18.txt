10.18 Step 18

The xargs command is typically used with the echo command, but it is useful with other commands as well. Use the xargs command to send the output of the find command to the grep command in order to search for all files with the .txt extension that contain the re characters:

find -name "*.txt" | xargs grep "re"

sysadmin@localhost:~$ find -name "*.txt" | xargs grep "re"
